************************************************************************************************************************
*                                                      Shell Basic Operators
************************************************************************************************************************

There are 5 types Operators
1. Arithmetic Operators
2. Relational Operators
3. Boolean Operators
4. String Operators
5. File Test Operators




************************************************************************************************************************
*                                                     File Test Operators
************************************************************************************************************************
------------------------------------------------------------------------------------------------------------------------
-d location                :   Checks Directory Existence
------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------
-f fileName                :   Checks file Existence
------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------
-e fileName                :   Check if file exists
------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------
-r fileName                :   Checks if file is readable
------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------
-w fileName                :   Check if file is writable
------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------
-x fileName                :   Check if file is executable
------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------
-s fileName                :   Check if file has size
------------------------------------------------------------------------------------------------------------------------
